Git删除 .DS_Store

 uyaki 收录于 git
2020-02-11 2020-02-11 约 41 字 预计阅读 1 分钟 
  1. cd 到项目路径
  2. 执行 
    $ find . -name .DS_Store -print0 | xargs -0 git rm -f --ignore-unmatch
  3. 插入到.gitignore 
    $ echo .DS_Store >> .gitignore
  4. git push
