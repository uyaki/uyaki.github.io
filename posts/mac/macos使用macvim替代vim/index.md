# MacOS使用MacVim替代vim


&lt;!--more--&gt;
```bash
## 安装macvim
brew install macvim

## 编辑配置文件
vim ~/.zshrc

## 添加
alias vi=vim
alias vim=mvim
alias mvim=&#39;/usr/local/bin/mvim -v&#39;

## 生效
source ~/.zshrc
```
