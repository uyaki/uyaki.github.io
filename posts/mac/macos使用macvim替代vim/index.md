# MacOS使用MacVim替代vim


<!--more-->
```bash
## 安装macvim
brew install macvim

## 编辑配置文件
vim ~/.zshrc

## 添加
alias vi=vim
alias vim=mvim
alias mvim='/usr/local/bin/mvim -v'

## 生效
source ~/.zshrc
```
