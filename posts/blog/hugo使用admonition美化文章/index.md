# Hugo使用admonition美化文章


&lt;!--more--&gt;

hugo的LoveIt，FixIt主题都集成了admonition shorcode功能，可以比较方便的进行使用，语法和样式效果如下。

## 注意

```markdown
{{&lt;/* admonition note */&gt;}}
一个 **注意** 横幅
{{&lt;/* /admonition */&gt;}}
```

{{&lt; admonition note&gt;}}
一个 **注意** 横幅
{{&lt; /admonition &gt;}}
