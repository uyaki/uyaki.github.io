  • 方式二（Mac重启后失效）：

方式一：

  1. 修改pem权限；

    1

    sudo chmod 600 key.pem

  2. Mac OS 连接服务器使用 PEM 文件；

    1

    ssh -i key.pem root@IP

方式二（Mac重启后失效）：

  1. 使用ssh-add添加key文件；

    1

    ssh-add -k key.pem

  2. 登陆；

    1
2

    ssh root@IP
ssh -p xxxx root@IP
