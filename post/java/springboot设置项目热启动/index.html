文章目录

  • 导入jar包

导入jar包

1

'org.springframework.boot:spring-boot-devtools'

添加配置项

1
2
3

devtools: #热部署
    restart:
      enabled: true

在idea开启

  1. 如下图设置 Build，Execution...->Compiler，勾选 Build project automatically
  2. 快捷键 commond+option+shift+'/'，弹出Maintenance界面
  3. 点击Registry, 勾选compiler.automake.allow.when.app.running
  4. 重启项目
