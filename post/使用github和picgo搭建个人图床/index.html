Github操作

新建repository作为图床地址

创建token

  1. 访问 https://github.com/settings/tokens，点击Generate new token；
  2. 把 repo 的勾打上即可。然后翻到页面最底部，点击 Generate token 的绿色按钮生成 token；
  3. 复制生成的token；

PicGo配置

安装

  1. 从PicGo官方下载最新版本；
  2. 使用jsDelivr进行CDN加速，设置自定义域名如下https://cdn.jsdelivr.net/gh/uyaba/pic-cloud；
